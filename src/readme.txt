The sources of NetCheat:

main.c: 	Initializes GS, loads settings, loads modules, and starts the server
gui.c:		Contains all the functions for PNG loading, VRAM uploading, and the menu. Taken from uCL-LiveDebug v4 (Original by Berion).
timer.c:	A place holder for a future use... Imported from uCL-LiveDebug v4
qwerty.c:	Contains module from CogSwap along with a method to load it into the IOP
